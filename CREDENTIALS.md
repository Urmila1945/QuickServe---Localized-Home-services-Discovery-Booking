## 🔧 Troubleshooting

**Network Error?**
- Backend not running
- Run `FIX.bat` then `START.bat`
- Wait 10-15 seconds

**MongoDB Error?**
- Run `START_MONGODB.bat`
- Or: `net start MongoDB` (as Admin)

**Port in Use?**
- Run `STOP.bat`
